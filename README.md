# Система управления личными финансами

Backend-приложение для управления личными финансами с возможностью учета доходов и расходов, установки бюджетов по категориям и получения финансовой статистики.

## Основные возможности

### Управление финансами
- Добавление доходов и расходов с указанием категорий
- Автоматический расчет текущего баланса
- Учет операций с датой и временем

### Бюджетирование
- Установка бюджетов на категории расходов
- Отслеживание оставшегося лимита по категориям
- Автоматические оповещения о превышении бюджета

### Отчетность и аналитика
- Общая статистика по доходам и расходам
- Детализация по категориям
- Отчеты за произвольные периоды
- Экспорт данных в CSV, JSON и текстовые форматы

### Система оповещений
- Уведомления о превышении бюджета
- Предупреждения при 80% использовании лимита
- Оповещения о низком балансе
- Уведомления о перерасходе (расходы > доходы)

### Управление данными
- Автоматическое сохранение данных при выходе
- Загрузка данных при авторизации
- Резервное копирование
- Экспорт/импорт данных

## Установка и запуск

### Требования
- Java 17 или выше
- Maven 3.6 или выше

### Сборка проекта

# Сборка проекта с Maven
mvn clean compile

# Создание исполняемого JAR-файла
mvn package

# Запуск из собранного JAR
java -jar target/personal-finance-manager-1.0.0.jar

# Запуск всех тестов
mvn test

# Очистка проекта
mvn clean
